# Weather App (Python + OpenWeatherMap API)

## Overview
A simple Python console application that fetches real-time weather data using the OpenWeatherMap API.  
It displays the current temperature, humidity, and weather conditions for any city entered by the user.

## Features
- Fetches live weather data from OpenWeatherMap
- Shows temperature (°C), humidity (%), and conditions (e.g., clear sky, rain)
- User can input any city name
